Expensave is a free, self-hosted expense tracker for personal and family budget management. No subscriptions. No cloud. Your server, your rules.

Features

Everything you need to manage finances

Built for individuals and families who want full control over their financial data.

👥

Multi-user support

Invite family members and share expense calendars. Everyone stays on the same page.

📅

Unlimited calendars

Create separate calendars for different budgets — household, travel, personal spending.

🔁

Recurring expenses

Set up daily, weekly, monthly, or yearly recurring transactions. Never miss a bill.

🏦

Bank statement import

Import transactions from your bank in various formats. Supports multiple financial institutions.

📊

Reports & insights

Visualise spending habits and income with clear, actionable reports.

📱

Mobile PWA

Install on iOS or Android directly from the browser. No app store required.

🔒

Your data, your server

Self-hosted means zero third-party access to your financial data. GPL-3 licensed.

🐳

Docker-first

One command to install. Runs anywhere Docker runs — VPS, home server, Raspberry Pi.

Installation

Up and running in minutes

Three ways to install — pick what works for you.

⭐ Recommended

Automated installer

Interactive bash script. Checks requirements, configures settings, and starts the app.

curl -fsSL https://raw.githubusercontent.com/algirdasc/expensave/main/install.sh | sudo bash
Docker Compose

Manual setup

Download the compose file and configure your own .env. Full control.

docker compose up -d

See full instructions →

One-click

Deploy on Hostinger
